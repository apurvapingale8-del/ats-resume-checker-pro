# ATS Resume Checker Pro

ATS Resume Checker Pro is a free online tool that helps job seekers compare their resumes with job descriptions and improve their ATS (Applicant Tracking System) compatibility.

## Features

* ATS Score Calculation
* Matching Skills Detection
* Missing Skills Identification
* Resume & Job Description Word Count
* Improvement Recommendations
* Mobile Responsive Design
* Fast and Free to Use

## Technologies Used

* HTML
* CSS
* JavaScript
